FAQ


How do I upload my data?

Choose your file. Pick your file format. Press Upload New Dataset. Wait a few mins then refresh the page and you should see it in pick Dataset options.


What is the expected file format?

Gustavo's format accepts .fa.sorted.annot.gaps.category files. Please ensure your file has no headers or other errors in it before uploading.

Bob's file format accepts his .tile.counts output file as is


I uploaded my file and can see it in the drop down but the vis isnt loading.

There was likely an error in converting your file to the vis file format. Most often this error occurs if one of the required values is null. Comb through your file to make sure there are no obvious mistakes. Let Aidan know and I can try and debug as well.
